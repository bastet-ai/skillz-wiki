# Launching Skillz Wiki

Skillz Wiki replaces the older project framing with a simpler model: an offensive-security wiki that also produces installable skills for agents.

## What changed

- the site theme and page mechanics were rebuilt for the Skillz Wiki format
- tool-specific content moved under **Skills**
- the public taxonomy now emphasizes **Recon** and **Exploit Paths**
- older alert and mitigation-oriented material is no longer the main navigation surface

## Initial skill set

- [HTTP Probing with httpx](../skills/httpx.md)
- [DNS Enumeration](../skills/dns-enumeration.md)
- [Nmap Scanning](../skills/nmap-scanning.md)

## Next steps

- expand the recon skill catalog to additional operator tools
- add more proven exploit paths that can be replayed with minimal-impact proofs
- keep skills compact enough to export directly into agent packages
