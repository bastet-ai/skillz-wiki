# Launching Skillz Wiki

Skillz Wiki replaces the older project framing with a simpler model: a security-tooling wiki that also produces installable skills for agents.

## What changed

- the site theme and page mechanics were rebuilt for the Skillz Wiki format
- tool-specific content moved under **Skills**
- supporting material is organized as playbooks, checklists, best practices, notes, and blog updates

## Initial skill set

- [DNS Enumeration](../skills/dns-enumeration.md)
- [Nmap Scanning](../skills/nmap-scanning.md)

## Next steps

- expand the skill catalog to additional security tools
- add more playbooks that connect tool output across phases
- keep skills compact enough to export directly into agent packages
